## Recommended Reading Order

1. Open URL Cheat Sheet for quick endpoint lookup.
2. Start with Getting Started for first run setup.
3. Read How It Works to understand architecture and routing.
4. Use Operations Guide for day-to-day commands.
5. Use Troubleshooting when something fails.
6. Use Documentation Checklist before and after infra changes.

## Scope

These docs describe local development infrastructure only.
Production hardening, TLS, HA, backup strategy, and SSO design are out of scope.

## Documentation Rule

Any infra change is incomplete until the affected document in this folder is updated.
